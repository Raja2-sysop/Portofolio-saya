Tentang Saya

Ini adalah Website Portofolio sederhana saya yang dibuat menggunakan Bootstrap

Saya adalah lulusan MAN 1 (Model) Lubuklinggau yang memiliki minat tak terhingga dalam pengembangan ilmu pengetahuan dan jurnalistik. Sejak SMA, saya aktif di berbagai kegiatan, terutama di bidang kepenulisan, yang dibuktikan dengan pencapaian prestasi karya tulis ilmiah di tingkat kabupaten hingga nasional. Selain itu, pengalaman saya sebagai wakil ketua dalam organisasi sekolah telah melatih saya untuk bekerja sama dalam tim secara efektif serta mengasah jiwa kepemimpinan yang mumpuni.

Saya juga memiliki minat yang besar di bidang teknologi, khususnya dalam mempelajari sistem operasi Linux (Ubuntu) untuk kebutuhan DevOps, serta bahasa pemrograman seperti Python untuk data science dan C untuk pengembangan program dasar dan sistem.
